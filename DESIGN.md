# Design System

## 1. Product Intent

`time_is_gold__` is a minimal deployment starter. The interface should feel direct, calm, and lightweight.

## 2. Color Tokens

- `--color-page`: `#f8f5ef`
- `--color-text`: `#1f2933`

## 3. Typography Tokens

- `--font-sans`: `Arial, Helvetica, sans-serif`
- `--text-title`: `clamp(2.25rem, 8vw, 5rem)`
- `--text-body`: `1.125rem`
- `--leading-title`: `0.95`
- `--leading-body`: `1.6`

## 4. Spacing Tokens

- `--space-page-inline`: `16px`
- `--space-title-gap`: `12px`
- `--content-max`: `680px`
- `--copy-max`: `34rem`

## 5. Components

- `main`: centered content container with a constrained width.
- `h1`: large title using the title scale.
- `p`: readable supporting copy using the body scale.

## 6. Responsive Behavior

The page centers content vertically and horizontally. The content width is constrained on desktop and leaves `16px` inline space on narrow screens.

## 7. Accessibility

Use semantic HTML, a Korean language attribute, viewport metadata, and sufficient text/background contrast.
